OpenAPI

  • open swagger https://editor.swagger.io/
  • download api documentation openapi.yaml
  • "Import file" in Swagger Editor
  • view the api
  • "Generate Client" by Swagger Editor

Precautions

  • By default, "POST" is not supported.
  • How to support
  • 1. open conf/conf.json
  • 2. set the value of "private_address" to "127.0.0.1:9090"
  • 3. restart govm.exe
